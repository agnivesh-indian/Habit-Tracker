# Habit Tracker

A simple, single-file, static web application for tracking habits and viewing statistics.

## Project Overview

This is a lightweight, zero-dependency web application that allows users to track their habits and gain insights into their progress. All the code (HTML, CSS, and JavaScript) is contained within a single `index.html` file, making it incredibly easy to run and deploy.

## Features

-   **Descriptive Statistics:**
    -   Mean Completion Rate
    -   Consistency Score (Standard Deviation)
    -   30-Day Trend Velocity
-   **Correlation Analysis:** See how completing one habit affects another.
-   **Simple UI:** A clean, functional interface for viewing all habits and diving into the details of a specific habit.

## Tech Stack

-   **HTML5**
-   **CSS3**
-   **JavaScript (ES6+)**

## How to Use

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/agnivesh-indian/Habit-Tracker.git
    ```
2.  **Open the file:**
    Navigate to the `Habit-Tracker` directory and open the `index.html` file in your web browser.

That's it! There are no dependencies to install or build steps to run.

## Deployment

You can deploy this application to any static web hosting service, such as GitHub Pages, by simply uploading the `index.html` file.
